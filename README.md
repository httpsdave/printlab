# PrintLab

PrintLab is a modern, web-based fingerprint analysis application that utilizes machine learning to predict a person's likely blood type based on a scanned fingerprint image. The application performs all predictive modeling efficiently and privately in the browser using TensorFlow Lite, ensuring sensitive biometric data never leaves the user's device. 

The project features a sleek, responsive interface built with React, Tailwind CSS, and sophisticated micro-interactions, providing a premium analytical experience.

## Setup & Development

To run this project locally:

1. Clone the repository:
   ```bash
   git clone https://github.com/httpsdave/printlab.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```
